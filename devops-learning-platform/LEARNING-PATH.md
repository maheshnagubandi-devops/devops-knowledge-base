# Learning Path

## Beginner
- Core Linux fundamentals
- Basic Docker and container concepts
- Intro to GitHub Actions and CI/CD
- Basic AWS services and infrastructure concepts

## Intermediate
- Kubernetes and container orchestration
- Infrastructure as Code with Terraform
- Observability, monitoring, and alerting
- Security fundamentals and DevSecOps awareness

## Advanced
- ArgoCD and GitOps workflows
- Cloud architecture patterns on AWS
- Platform engineering best practices
- System design for reliability, scalability, and security

## Interview Preparation Strategy
- Review concept summaries and cheat sheets
- Practice use cases and scenario-based questions
- Build mini projects to demonstrate applied knowledge
- Keep role-specific prep for DevOps, SRE, Platform, and Cloud interviews
