How to publish

  1. Push this branch to GitHub.
  2. The workflow will deploy the content from github-docs/ to the gh-pages branch.
  3. Open the repository on GitHub and check the Actions tab.
  4. After the workflow finishes, your site will be available from GitHub Pages.
