# DevOps Learning Platform

A structured knowledge base and interview preparation platform for DevOps, SRE, Platform Engineering, and AWS infrastructure roles.

This repository is built to serve as:

- A comprehensive learning path with beginner, intermediate, and advanced tracks
- A teaching resource with concept guides, use cases, and diagrams
- An interview prep library with cheat sheets, question banks, and scenario notes
- A project catalogue for practical hands-on learning
- A reusable template library for automation and CI/CD workflows

## Top-level structure

- glossary/ - key DevOps and cloud terms
- assets/ - diagrams, screenshots, and architecture visuals
- linux/ - deep Linux learning path with labs and cheat sheets
- docker/, kubernetes/, terraform/, aws/, jenkins/, github-actions/, helm/, argocd/ - core stack topics
- sre/, monitoring/, networking/, security/, system-design/ - operational and architecture topics
- projects/ - guided project ideas and implementations
- interview-preparation/ - role-focused question banks and memory aids
- troubleshooting-runbooks/ - production troubleshooting guides
- reusable-templates/ - code templates for common Cloud and automation patterns
- scripts/ - helper scripts for automating study or demo setup

> Note: this repo is optimized for teaching, interview preparation, cheat sheets, and documentation. Full project repositories are often better stored separately so they can be shared directly with interviewers and kept clean from teaching content.
